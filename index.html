前言

我认为你需要好好规划一下暑假的生活了，不能这样无所事事下去，定下来一个战略目标，有所取舍，然后让发展的问题在发展中得到解决。对，首先你需要明确的是暑假内你的目标是什么，之前反思过，今后很长一段时间内你的目标都是在实践中提高自身处理问题的能力，那么分解到这个暑假，你要从哪些方面提高自己的能力呢？

编程

编程的角度来看，在暑假计划中提到，是语言（C++、Python、C#）和常用工具（Git、Github、Visual Studio、Visual Studio Code、Cmake）的使用。如何将目标落实到实践中去，阅读开源项目源码，在开源项目基础上，自己试着改造几个简单的项目。

处理一般问题

找实习、做家教、做实践

深入思考问题

数学建模、数据库增量备份

小结

以上三个方面讨论了可能会遇到的三种问题，第一种要求专，第二种要求宽，第三种要求深。要以专的为主线，就好比打磨自身的武器，以宽的作为日常，就好比每天都要处理的简单事项，以深的作为挑战，就好比打BOSS。专一的是能力本身，其他二者更像是磨砺的手段。劳动本身和提高劳动能力的方式。

谈一谈人生

人的诞生显然不是预先就带着意义的，即使你认为有些人天生就担任着使命，那这个使命也是他人强加的。自己生命的意义，由他人赋予，这谈何意义。所以人的意义是在成长的过程中，在主观与客观的对立统一中丰富和发展起来的，亦即人的意义就在于人本身，人的独特性。一方面主客观的对立统一推动人格、人性的发展，另一方面主客观的对立统一又会泯灭人本身的独立性。处理好主观与客观的关系，就是处理好个人与外界的关系，而主观与客观通过实践连接，个人与外界通过劳动连接。那么劳动存在的意义就是在主客观的对立统一中推动人本身的发展，另一方面也会为客观世界的改造带来转变。

唯物的角度来看，人的独特性就体现在劳动上，而非空想。目标就由意义决定，目标就是不让自己的劳动被异化，总是让劳动成为完善人格的手段，而非生存的手段或是其他。

劳动的能力！主观客观统一的能力是当前阶段的目标

2022-06-30

暑假

目标

  1. 提高人际交往能力，处理事情的反应能力，面对事情的沉稳性
  2. 提高实际编程能力，真正投入到现实生活的项目开发中

解释

  1. 处理复杂事情的能力是在社会中生活的人必备的能力。
    社会上不再像学校中那样在单一的环境中做单一的事情，诚然，我只有在处理一件事情的时候才能把它做好，但这不意味着我可以一辈子仅在处理简单的事情。
    在复杂的环境中提高自己处理复杂问题能力是我必须通过的一场试炼。
    处理复杂的问题并不意味着眉毛胡子一把抓，乱做一气，这种能力的核心在于对复杂问题的分解，和应对事情的本能与沉稳。
    事实上，现实生活中根本不存在纯粹的简单事情，在人们眼中能力强大的人才本质上就是拥有洞悉问题本质，并将其分解的能力。
    如果没有这种能力，在面对现实问题时，只能依靠现有的框架，或是反复练习养成的习惯。而且必须是几乎一模一样的问题
  1. 编程这件事在我看来是现代社会最强大的武器，或者说它是可以调用人类智慧结晶的最强力手段。
    通过计算机的强大计算能力实现人类无法完成，很难完成，不愿完成的事情。

任务

  1. 复杂问题：
    1. IE亮剑
    2. 实习机会
    3. 社会实践机会 隐藏任务：了解小罗同学
    4. 数据库增量备份论文
    5. 数模准备
  1. 编程：
    1. 现实中的工具的应用，git、GitHub、visual studio，visual studio code
    2. 现实中需要灵活应用的语言，python，c#， c++
    3. 真正在生活中开发一款软件，可以从游戏入手，再到数据库管理系统（DBMS）的设计

计划

原则

  1. 每天都要有解决复杂问题的训练，也要有编程能力（信息化工具使用能力）的提高
  2. 一切都要落实到实践，即实现主观与客观的统一。
